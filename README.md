# run-visual.nvim

A neovim plugin to run your selection text/code on visual mode.

## Setup

- lua

```lua
{
  "lost22git/run-visual.nvim",
  lazy = false,
  opts = {},
}
```

- fennel

```fennel
(1 "lost22git/run-visual.nvim" :lazy false :opts {})
```
